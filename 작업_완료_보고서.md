## 🎯 핵심 개선 사항

### 1. 해시태그 룰 시스템
- **블랙리스트**: Etsy, Minne, Creema 등 경쟁사 태그 자동 필터링
- **화이트리스트**: #아이디어스, #idus, #アイディアス 등 브랜드 태그 필수 포함
- **카테고리별 권장 태그**: 도자기, 가죽, 캔들, 주얼리 등

### 2. 통합 콘텐츠 스튜디오
- 소셜 콘텐츠 + CRM 카피 동시 생성
- 결과 탭 전환으로 한 화면에서 확인
- A/B 테스트 카피 제공 (감성형/긴급형)

### 3. CRM 확장성
- **작품 기반** (기본): 선택한 작품 구매 유도
- **기획전 기반** (확장): 기획전 URL 입력 → 테마 강조 CRM
- **작가 홈 기반** (확장): 작가 URL 입력 → 작가 스토리 CRM

### 4. UI/UX 개선
- SUITE 폰트로 idus 브랜드 일관성
- 브랜드 오렌지(#FF6B35) 포인트 컬러
- Framer Motion 애니메이션
- 로딩 상태 시각화
