## 1. 핵심 설계 원칙

### 🎯 UX 원칙

| 원칙 | 설명 |
|------|------|
| **기능 노출 최대화** | 모든 기능은 숨기지 않고 명확히 보이도록 배치 |
| **업무 흐름 일치** | 실제 마케터 업무 순서와 UI 흐름 일치 |
| **원클릭 완성** | 핵심 작업은 최대 3클릭 이내 완료 |
| **시각적 피드백** | 모든 상태 변화에 즉각적 피드백 제공 |
| **컨텍스트 유지** | 이전 단계 정보가 다음 단계에 자연스럽게 연결 |

### 📐 디자인 원칙

```
┌─────────────────────────────────────────────────────────────────┐
│  1. 일관된 카드 시스템                                           │
│     - 모든 정보는 카드 단위로 구성                                │
│     - 호버/선택 상태 명확히 구분                                  │
│                                                                 │
│  2. 명확한 시각적 계층                                           │
│     - Primary Action: 보라색 그라데이션 버튼                      │
│     - Secondary Action: 아웃라인 버튼                            │
│     - 정보 구분: 섹션 카드 + 디바이더                             │
│                                                                 │
│  3. 진행 상태 표시                                               │
│     - 선택된 작품 수, 생성 진행률 등 항상 표시                     │
│     - Breadcrumb 형태의 단계 표시                                │
│                                                                 │
│  4. 반응형 그리드                                                │
│     - 작품 카드: 반응형 4/3/2열 그리드                            │
│     - 결과 카드: 유연한 Masonry 레이아웃                          │
└─────────────────────────────────────────────────────────────────┘
```

## 5. Tab 2: 콘텐츠 생성 (Content Studio)

### 5.1 기능 목표

> **"선택한 작품 기반으로 소셜 콘텐츠 + CRM 카피를 한 화면에서 통합 생성"**

### 5.2 핵심 설계 포인트

```
1. 소셜 콘텐츠와 CRM은 하나의 탭에서 관리
   - 같은 작품 기반이므로 자연스럽게 연결
   - 탭 전환 없이 빠른 작업 가능

2. CRM 기본 모드: 작품 기반 (소개/구매독려)
   - 작품 탐색에서 선택한 작품 정보 자동 연결
   - 가장 많이 사용하는 케이스

3. CRM 확장 모드: 토글로 추가 옵션 노출
   - 기획전 CRM
   - 작가 홈 CRM
   - 필요할 때만 펼쳐서 사용
```

### 5.3 전체 레이아웃


### 9.2 API 엔드포인트 구현

```typescript
// api/crawl/search.ts
import puppeteer from 'puppeteer-core';
import chromium from '@sparticuz/chromium-min';

export const config = {
  maxDuration: 30, // 30초 타임아웃
};

interface ProductSearchResult {
  id: string;
  title: string;
  price: number;
  originalPrice?: number;
  discountRate?: number;
  image: string;
  artistName: string;
  rating: number;
  reviewCount: number;
  url: string;
}

export default async function handler(req: Request) {
  if (req.method !== 'POST') {
    return new Response(JSON.stringify({ error: 'Method not allowed' }), { status: 405 });
  }

  const { keyword, sort = 'popular', page = 1 } = await req.json();

  if (!keyword || keyword.trim().length === 0) {
    return new Response(JSON.stringify({ error: 'Keyword is required' }), { status: 400 });
  }

  try {
    const browser = await puppeteer.launch({
      args: [...chromium.args, '--hide-scrollbars', '--disable-web-security'],
      defaultViewport: chromium.defaultViewport,
      executablePath: await chromium.executablePath(
        'https://github.com/nicholasjhenry/chromium/releases/download/v123.0.0/chromium-v123.0.0-pack.tar'
      ),
      headless: true,
    });

    const browserPage = await browser.newPage();

    // User Agent 설정 (봇 탐지 우회)
    await browserPage.setUserAgent(
      'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36'
    );

    // 불필요한 리소스 차단 (속도 향상)
    await browserPage.setRequestInterception(true);
    browserPage.on('request', (request) => {
      const resourceType = request.resourceType();
      if (['stylesheet', 'font', 'media'].includes(resourceType)) {
        request.abort();
      } else {
        request.continue();
      }
    });

    // 검색 페이지 접근
    const sortMap: Record<string, string> = {
      popular: 'popular',
      newest: 'recent',
      price_asc: 'price_asc',
      price_desc: 'price_desc',
      rating: 'rating',
    };

    const searchUrl = `https://www.idus.com/search?keyword=${encodeURIComponent(keyword)}&order=${sortMap[sort] || 'popular'}`;
    
    await browserPage.goto(searchUrl, { 
      waitUntil: 'domcontentloaded',
      timeout: 15000,
    });

    // 상품 로드 대기
    await browserPage.waitForSelector('[data-product-id]', { timeout: 10000 }).catch(() => {});

    // 스크롤하여 더 많은 상품 로드
    await autoScroll(browserPage, 3);

    // 상품 데이터 추출
    const products: ProductSearchResult[] = await browserPage.evaluate(() => {
      const items = document.querySelectorAll('[data-product-id]');
      
      return Array.from(items).slice(0, 24).map((item) => {
        const getId = () => item.getAttribute('data-product-id') || '';
        const getTitle = () => item.querySelector('.product-title, .item-title, h3')?.textContent?.trim() || '';
        const getPriceText = () => item.querySelector('.price, .item-price')?.textContent || '0';
        const getImage = () => {
          const img = item.querySelector('img');
          return img?.src || img?.getAttribute('data-src') || '';
        };
        const getArtist = () => item.querySelector('.artist-name, .seller-name')?.textContent?.trim() || '';
        const getRating = () => {
          const ratingText = item.querySelector('.rating, .star-rating')?.textContent || '0';
          return parseFloat(ratingText) || 0;
        };
        const getReviewCount = () => {
          const reviewText = item.querySelector('.review-count')?.textContent || '0';
          return parseInt(reviewText.replace(/[^0-9]/g, '')) || 0;
        };
        const getUrl = () => {
          const link = item.querySelector('a');
          return link?.href || `https://www.idus.com/w/product/${getId()}`;
        };

        const priceText = getPriceText().replace(/[^0-9]/g, '');
        
        return {
          id: getId(),
          title: getTitle(),
          price: parseInt(priceText) || 0,
          image: getImage(),
          artistName: getArtist(),
          rating: getRating(),
          reviewCount: getReviewCount(),
          url: getUrl(),
        };
      }).filter(p => p.id && p.title);
    });

    await browser.close();

    return new Response(JSON.stringify({ 
      products,
      total: products.length,
      keyword,
      sort,
    }), {
      status: 200,
      headers: { 'Content-Type': 'application/json' },
    });

  } catch (error) {
    console.error('Crawling error:', error);
    return new Response(JSON.stringify({ 
      error: 'Failed to fetch products',
      message: error instanceof Error ? error.message : 'Unknown error',
    }), { status: 500 });
  }
}

async function autoScroll(page: any, scrollCount: number = 3) {
  for (let i = 0; i < scrollCount; i++) {
    await page.evaluate(() => {
      window.scrollTo(0, document.body.scrollHeight);
    });
    await new Promise(resolve => setTimeout(resolve, 500));
  }
}
```

### 9.3 클라이언트 서비스

```typescript
// services/idusService.ts
import { useState, useCallback } from 'react';

export interface IdusProduct {
  id: string;
  title: string;
  price: number;
  image: string;
  artistName: string;
  rating: number;
  reviewCount: number;
  url: string;
}

export interface SearchParams {
  keyword: string;
  sort?: 'popular' | 'newest' | 'price_asc' | 'price_desc' | 'rating';
}

export async function searchIdusProducts(params: SearchParams): Promise<IdusProduct[]> {
  const response = await fetch('/api/crawl/search', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(params),
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message || '검색에 실패했습니다');
  }

  const data = await response.json();
  return data.products;
}

// React Hook
export function useIdusSearch() {
  const [products, setProducts] = useState<IdusProduct[]>([]);
  const [isLoading, setIsLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);

  const search = useCallback(async (params: SearchParams) => {
    setIsLoading(true);
    setError(null);
    
    try {
      const results = await searchIdusProducts(params);
      setProducts(results);
    } catch (e) {
      setError(e instanceof Error ? e.message : '검색 중 오류가 발생했습니다');
      setProducts([]);
    } finally {
      setIsLoading(false);
    }
  }, []);

  const clear = useCallback(() => {
    setProducts([]);
    setError(null);
  }, []);

  return { products, isLoading, error, search, clear };
}
```
